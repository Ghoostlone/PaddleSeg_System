# 3D 肺部分割 Web 系统

![项目演示图](示例图片的链接)

## 项目简介

本项目是一个用于 3D 腹部内脏分割的 Web 系统，基于 PaddleSeg 实现。该系统允许用户上传医学图像，进行自动的付脏分割，以帮助医疗专业人员更好地分析。

## 特性

- 使用深度学习技术进行 3D 肺部分割。
- 用户友好的 Web 界面，允许医疗专业人员上传图像并查看结果。
- 允许病人查看自己腹脏的3D建模
- 支持自动化和半自动化分割选项，以满足不同需求。
